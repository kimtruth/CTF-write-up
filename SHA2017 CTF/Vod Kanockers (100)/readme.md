## Vod Kanockers (100)

> The name is Kanockers. Vod Kanockers.

 먼저 문제의 html 소스를 보면 knock knock을 발견할 수 있다.

 ```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>The name is Kanockers. Vod Kanockers</title>
  </head>
  <body>
    <!-- *Knock Knock* 88 156 983 1287 8743 5622 9123 -->
    <img src="vod.jpg" />
  </body>
</html>
```

순서대로 knock knock해주면 답이 나오게 된다.
